# Home

`/wiki-query` — read this first to orient queries against project context.

## Project at a glance

We're testing whether LLMs and agentic systems can serve as computational
engines for predicting cellular behavior. The task: predict perturbation
outcomes in macrophages.

Three tracks (canonical detail in
[`../challenge.md`](../challenge.md)):
- **Track A** — Prompt-only, fixed LLM, no tools.
- **Track B** — Agentic tool-use, fixed LLM.
- **Track C** — Fine-tuning, open model < 10B.

We're likely entering A and/or B.
